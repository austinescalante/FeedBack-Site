
#   Feedback Site

##  This is a Survey Web app based on a BWM dealer service. The user will be prompted to respond on feedback and rate their service.

##  Create a virtual environment 
```
pipenv shell
````


##  Start the server and a port can be specified.
````
python app.py

````
